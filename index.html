Graph Theory Project

A group project I worked on with a partner for a graph theory class. The program is a gamified version of disease spread, where users can choose three diseases, each with a different means of infection(airborne, waterborne, and spread by animals). The project uses a map that has a graph with 10 cities on it acting as vertices. The program randomly generates edges of the three infection types as well as weights for the edges. I then used Prim's algorithm to find a minimum spanning tree for the graph, which is diplayed by pink edges and vertices.
